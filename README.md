# HachiShifter

基于HiFiShifter，HachiShifter将试图增加更多能简化人力制作和提升效果的功能，特别是注重于素材管理功能。

**由于本人目前生活和学业比较忙，项目目前处于画饼状态，更新会缓慢**
##画饼中:
- [ ] 更好的素材管理系统，用于方便查找合适的采样。
- [ ] 支持根据标注文件实现节奏对齐。
- [ ] 支持读取utau素材的oto风格标注。
- [ ] 支持直接导入ust/ustx/vsqx进行参考或进行粗略调音。
- [ ] 自动素材标注（应作为单独程序发布）。
- [ ] 自动素材匹配（当一个音有多个采样时候根据音高和时长自动选择最合适的采样）
- [ ] 更好的拉伸算法。
